# Frontend_of_Accenture_Clone
A simple, static multi-page website for **Acceture** (a fictional IT consulting company inspired by Accenture).

## 📋 Project Description

This is a basic HTML-based website showcasing a company's:
- Home page with company vision and contact details
- Products/Services offered
- Branch locations across India
- Customer Feedback form

The website is built using pure HTML and CSS (no backend or JavaScript frameworks).

## 📁 Project Structure
Acceture/
├── index.html          # Home Page

├── product.html        # Services / Products Page

├── branches.html       # Our Branches Page

├── feedback.html       # Feedback Form Page

└── README.md


## 🌐 Pages Overview

### 1. **Home Page** (`index.html`)
- Company logo/name: **ACCENTURE**
- Navigation menu (Products, Branches, Feedback)
- Company address and contact information
- Vision statement

### 2. **Products/Services** (`product.html`)
Showcases the main service offerings:
- Cloud Services
- AI & Analytics
- Cyber Security
- Consulting

### 3. **Branches** (`branches.html`)
Lists company branches in major Indian cities:
- Mumbai (Navi Mumbai, Maharashtra)
- Bangalore (Electronic City, Karnataka)
- Hyderabad (Hitech City, Telangana)
- Pune (Hinjewadi IT Park)

### 4. **Feedback** (`feedback.html`)
Simple feedback form including:
- Name, Mobile, Email
- Education level
- Gender
- Feedback type
- Comments

## 🚀 How to Run

1. Download or clone the project folder.
2. Open any of the `.html` files in your web browser (Chrome, Firefox, Edge, etc.).
3. No installation or server required — fully static website.

**Recommended:**
- Start by opening `index.html`

## 🛠️ Technologies Used

- HTML5
- CSS3 (inline + basic styling)
